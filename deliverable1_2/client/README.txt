===========================================
        FashionKart - E-Commerce Site
===========================================

Description:
------------
FashionKart is a front-end e-commerce web application that displays trendy fashion products like T-shirts, hoodies, jeans, shoes, and accessories. The website is built using HTML, CSS, and JavaScript and includes a hero section, product listings, contact form, admin panel UI, and responsive design.

Folder Structure:
-----------------
project-root/
│
├── index.html              --> Main homepage
├── admin.html              --> Admin panel interface
├── CSS/
│   └── style.css           --> All styling and responsive layout
├── JS/
│   └── script.js           --> JS functions (e.g., cart actions, form validation)
├── assets/
│   └── image/              --> All product and UI images
│       ├── background.jpg
│       ├── t-shirt.jpg
│       ├── Winter-Hoodie.jpg
│       ├── Blue-Denim-Jeans.jpg
│       ├── Casual-Jacket.jpg
│       ├── White-Sneakers.jpg
│       ├── Canvas-Backpack.jpg
│       ├── Street-Style-Cap.jpg
│       ├── Smart-Casual-Watch.jpg
│       └── user-avatar.png

Features:
---------
- Attractive hero section with background image.
- Interactive product grid with "Add to Cart" buttons.
- Responsive design using Flexbox and media queries.
- Clean navigation bar with Sign In / Sign Up buttons.
- Admin login and product management UI (visual only).
- Contact form with validation.
- Header and footer with consistent branding.

Important Notes:
----------------
1. Make sure the folder is named 'assets' (not 'assests').
2. Image paths in both HTML and CSS should reference:
   -> assets/image/your-image-name.jpg
3. Run the website by simply opening index.html in your browser.
4. No server or backend is required; this is a static front-end project.

How to Use:
-----------
1. Download or clone the project folder.
2. Rename 'assests' folder to 'assets' if needed.
3. Open index.html in your browser.
4. Browse products, interact with the interface.
